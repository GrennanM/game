# Game 2048

## Run locally

- Install [Elixir](https://elixir-lang.org/install.html)
- Fetch dependencies:

```bash
mix deps.get
```

- Run server:

```bash
mix phx.server
```

- Navigate to endpoint: http://localhost:4000/game2048

- Play using the arrow keys to move tiles up, down, left and right.

## Tests

```bash
mix test
```
